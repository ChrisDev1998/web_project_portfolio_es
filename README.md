# Portafolio

Este proyecto es un portafolio personal desarrollado con **HTML y CSS**, siguiendo la metodología **BEM** para mantener el código escalable y fácil de mantener.  

## 🚀 Características
- Animaciones ligeras y transiciones gráficas para una experiencia fluida.
- Uso de **iconos SVG** optimizados para mejor rendimiento.
- Diseño responsivo con variantes de header (colapsado y expandido).
- Despliegue en **GitHub Pages** (con opción de dominio personalizado).

## 📂 Proyectos incluidos
1. **Proyecto A** – Landding Page
2. **Proyecto B** – Videojuego en proceso

## 🔧 Tecnologías
- HTML5
- CSS3 (Grid, Flexbox, pseudo-elementos animados)
- SVG
